PERTANYAAN NO 13-C :

13-C. Use nested layouts technique to break down this design into horizontal and vertical layouts.


JAWABAN NO 13-C :

Analisis layout satu baris email Gmail:
Desain satu baris email terdiri dari tiga bagian utama: gambar avatar, informasi email (seperti nama pengirim, judul, dan cuplikan), serta waktu.


Layout ini menggunakan layout bersarang (nested layout) sebagai berikut:

  1. Level 1 – Layout Vertikal (display: block):
    1. avatar (gambar di kiri)
    2. email-text (blok teks di tengah)
    3. time (jam di kanan)
      • Tiga elemen ini disusun secara horizontal menggunakan Flexbox.

  2. Level 2 – Layout Vertical di dalam email-text (display: block):
    1. sender (nama pengguna, bold)
    2. subject (judul pesan)
    3. snippet (cuplikan pesan, abu-abu)
      • Elemen-elemen ini disusun secara vertikal di dalam satu kolom.

Struktur Bersarang Secara Visual:

.email-item (flex horizontal) │ ├── .avatar (gambar bulat) ├── .email-text (vertical layout) │ ├── .sender │ ├── .subject │ └── .snippet └── .time (jam)


Contoh Kode:

          .email-item {
            display: flex;
            flex-direction: row;
          }
          .avatar {
            width: 55px;
            height: 55px;
            border-radius: 50%;
          }
          .email-text {
            display: flex;
            flex-direction: column;
          }
          .time {
            font-size: 12px;
          }


KESIMPULAN :

Dengan teknik layout bersarang, tampilan email disusun menggunakan kombinasi Flexbox (horizontal) untuk elemen utama dan layout vertikal biasa untuk isi teks, agar menyerupai tampilan aplikasi Gmail di iOS.
